# Install, Setup modules

```sh
$ brew install closure-compiler

$ git clone git@github.com:uupaa/Ruby.js.git
$ cd Ruby.js
$ npm install
```

# Minify

```sh
$ npm start

  or

$ node node_modules/uupaa.minify.js --keep --output ./lib/Ruby.min.js ./lib/Ruby.js
```

# Test

```sh
$ npm test
```
